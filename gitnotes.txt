Here is a consolidated guide from creating a repository to pushing your changes to GitHub:

Create a repository on GitHub named Typescript-exercises.

Clone the repository to local:


git clone https://github.com/your-username/Typescript-exercises.git
cd Typescript-exercises
Add your scripts to the local folder.

Check the status of your repository:

git status
Stage the changes:


git add .
Commit the changes:

git commit -m "new changes"
Push the changes to GitHub:


git push origin main
